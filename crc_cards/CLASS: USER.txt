CLASS: USER 

    - Responsibility
        - store user information on niubi system
        - override the setPassword method from legacy.Register interface

    - Collaborators
        - legacy.Register, which allows users to register
        - TableReader, where stores the user accounts info(users can register by choosing a new username,
        or log in by typing the correct username and password).
        - UserInfo, which help users to register or log in by checking between user info and TableReader
        - WelcomePartController, where users can register or log in.
