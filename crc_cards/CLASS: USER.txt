CLASS: USER 

    - Responsibility
        - allow users to register their accounts
        - store user information on niubi system
        - override the setPassword method from legacy.Register interface

    - Collaborators
        - legacy.Register
        - data.userinfo
        - TableReader, where stores the user accounts info(users can register by choosing a unique username,
        or log in by typing the correct username and password).
        - UserInfo, which checks if user can register or log in successfully.
        - WelcomePartController, where users can register or log in.
