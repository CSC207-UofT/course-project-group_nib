Entity Class: Entity.Note

    - Responsibility
        - Stores notes info such as "editable" "commentable" "referable" "changes" "comment" "reference"
        - Updates the note info when needed.

    - Collaborators
        - Child Classes CategoryA, CategoryB, CategoryC, CategoryD, CategoryE, which stand for different branches of courses.
        - Controller INotes, which edits the note info.
